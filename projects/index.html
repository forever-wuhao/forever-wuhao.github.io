以下是我参与或独立完成的一些项目，涵盖全栈开发、移动端、AI应用等多个领域。

携程训练营 · 酒店AI客服系统

时间： 2026/02 - 2026/04
角色： 全栈工程师
技术栈： SpringBoot · Vue 3 · Android · 高德地图 · 智谱AI GLM-4 · JWT · MySQL · Redis

基于 SpringBoot + Vue 3 构建的酒店预订全栈系统，集成高德地图与智谱AI GLM-4，实现酒店搜索、地图定位与AI智能客服。

核心亮点：

  • 前端集成高德地图API，实现酒店地理位置精准选择与可视化展示
  • Android客户端使用 Retrofit + OkHttp 封装网络层，实现 ViewPager2 + TabLayout 轮播图与多Tab酒店分类展示
  • 后端基于 SpringBoot + MyBatis-Plus 搭建高性能服务端，独立设计酒店、房型、订单等10+张核心数据表
  • 集成智谱AI GLM-4接口开发AI酒店助手，实现浮动按钮与聊天对话框，提供智能预订建议与流式对话交互
  • 实现基于JWT的统一身份认证与授权方案，封装Map类型统计信息返回，优化Android端数据展示效率

大白AI心理陪伴助手

时间： 2025/12 - 2026/04
角色： 全栈工程师
技术栈： Vue 3 · Capacitor · 腾讯混元AI · AI图像识别

基于 Vue 3 + Capacitor 构建的跨端心理健康应用，集成腾讯混元AI实现拍照情绪检测与智能心理咨询，为用户提供情绪管理与AI陪伴服务。

核心亮点：

  • 开发AI拍照情绪检测模块，用户拍照后调用AI接口实时分析情绪状态，生成个性化心理建议与干预方案
  • 基于Vue 3构建APP界面，实现心情记录、树洞对话、情绪日历等核心功能，使用Capacitor打包为跨端应用
  • 封装移动端API签名认证机制，解决腾讯混元AI接口在移动端的调用稳定性问题，保障AI服务可用性
